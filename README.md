# CashTrack v8

שינויים בגרסה זו:

- השדה "הכנסה נוספת" מוגדר במפורש כהכנסה נטו שנכנסת לחשבון.
- יעד מינימום בעו״ש הוסר.
- נוסף תקציב חודשי קבוע לבזבוזים (ברירת מחדל: 1,500 ₪).
- נוסף שדה "בזבוזים החודש עד עכשיו".
- יתרת תקציב = תקציב חודשי פחות בזבוזים שבוצעו.
- בזבוזים אינם מופחתים פעם נוספת מהעו״ש, כי הם בדרך כלל כבר כלולים בחיוב האשראי.
- נוסף צפי שינוי בעו״ש, כדי לראות אם החשבון צפוי לגדול או לקטון.
