# CashTrack v14

## ייבוא עסקאות מצילומי מסך

בלשונית **עו״ש** נוסף אזור **„ייבוא עסקאות מצילום מסך”**.

1. בוחרים צילום אחד או כמה צילומים מאפליקציית האשראי.
2. לוחצים **„זהה עסקאות בצילומים”**.
3. האפליקציה מנסה לזהות שם עסק, תאריך וסכום.
4. מתקבלת תצוגה מקדימה שאפשר לתקן לפני האישור.
5. בוחרים אילו עסקאות נכללות בתקציב הבזבוזים.
6. מאשרים את העסקאות המסומנות.

הזיהוי מתבצע בדפדפן במכשיר. התמונה אינה נשלחת לשרת של CashTrack. מנוע הזיהוי וקובצי השפה נטענים מהאינטרנט ולכן נדרש חיבור בזמן הזיהוי.

עסקת תשלומים מתאריך ישן יכולה להיספר בחודש החיוב שנבחר באפליקציה, תוך שמירת התאריך המקורי.

עסקה בעלת אותו חודש חיוב, תאריך, שם וסכום מזוהה ככפילות ולא מסומנת לייבוא כברירת מחדל.

זיהוי טקסט אינו מושלם ולכן חשוב לבדוק את התצוגה המקדימה.
